Job Search Web Application

Video Streaming with Quic Protocol

The aim of this project was to demonstrate the robustness of QUIC protocol on low bandwidth networks while streaming real time data. The performance of a QUIC socket was compared against the performance against a TCP socket on networks of variable bandwidths. Both C++ and Python were used for this project.

Twitter Sentiment Analysis

The main idea behind this project is to understand the general sentiment of twitter users about a topic based on their tweets. A live stream of tweets was fetched and using Kafka through twitter's API and was processed using Apache Spark and Python (PySpark). Natural Language processing techniques were then employed to fnd out the sentiments about a topic.
